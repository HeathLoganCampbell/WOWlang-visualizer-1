Commands

WOW: Increment value
wow: Decrement value
woW: Move register pointer to right
Wow: Move register pointer to left
wOw: Read input value to register
WoW: Output current register value
WOw: Open loop
woW: Close loop
woow: Add value from register to pointer position
WOOW: Copy value from register
w0w: Reset register value
